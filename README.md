The most compact & optimize UART0 and UART1 Bootloader for Nuvoton N76E003 & MS51FB9AE MCUs. Supports NuvoISP protocol.
Please use NuvoISP version 4.10 or earlier as later versions have a bug when running from the command line.
When selecting a 1KB size:
1. Does not support writing configuration bits
2. Does not support restoring the default clock frequency of 16 MHz
